## Try it now!

> requires Node >=14

## Usage

### Install

```
pnpm install
```

### Development

Just run

```bash
pnpm dev
```

### Build

To build the App, run

```bash
pnpm build
```

And you will see the generated file in `dist` that ready to be served.
